One-time setup: run this SQL in Supabase → SQL Editor:

1. Platform invoices table:
CREATE TABLE platform_invoices (id uuid PRIMARY KEY DEFAULT gen_random_uuid(), account_email text, account_user_id uuid, date date, due_date date, inv_num text, payment text, notes text, inv_total numeric(10,2) DEFAULT 0, inv_bal numeric(10,2) DEFAULT 0, service_lines jsonb, status_text text, failed_charge boolean DEFAULT false, deleted boolean DEFAULT false, past_due boolean DEFAULT false, to_print boolean DEFAULT false, to_email boolean DEFAULT false, created_at timestamptz DEFAULT now());

2. Recurring billing table:
CREATE TABLE platform_accounts (id uuid PRIMARY KEY DEFAULT gen_random_uuid(), user_id uuid, email text, plan text, monthly_amount numeric(10,2), billing_day int, active boolean DEFAULT true, created_at timestamptz DEFAULT now());
