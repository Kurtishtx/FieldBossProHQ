One-time setup — run this SQL in Supabase → SQL Editor:

CREATE TABLE IF NOT EXISTS email_alert_templates (id uuid DEFAULT uuid_generate_v4() PRIMARY KEY, user_id uuid REFERENCES auth.users(id) ON DELETE CASCADE, name text NOT NULL, body text, style int DEFAULT 1, created_at timestamptz DEFAULT now()); ALTER TABLE email_alert_templates ENABLE ROW LEVEL SECURITY; CREATE POLICY "own" ON email_alert_templates FOR ALL USING (auth.uid() = user_id);
