User Roles & Rights

Configure what each role can access on the desktop and mobile app.

One-time setup — run this in Supabase → SQL Editor:

CREATE POLICY "tenant_read_perms" ON role_permissions FOR SELECT USING (EXISTS (SELECT 1 FROM user_profiles WHERE id = auth.uid() AND tenant_id = role_permissions.user_id) OR auth.uid() = user_id);
